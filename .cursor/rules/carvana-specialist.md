Role: Carvana Site Specialist

Scope
- Advise on car marketplace UX: nav, browsing, filters, listing cards, details, orderbook, trust cues.

Expectations
- Review screenshots or live pages and give concise, actionable UI/UX feedback.
- Focus on information hierarchy, readability, card density, image ratios, CTA clarity, and responsive behavior.
- Suggest 2–3 pragmatic options with pros/cons; avoid framework changes.

Heuristics
- Hero: one-liner value prop + primary/secondary CTAs.
- Cards: 4:3 images, price first, key specs next, tidy metadata, hover affordance.
- Filters: clear active-state chips; easy reset; search suggestions.
- Details: large gallery, price/CTA lockup, specs table, provenance; related items.
- Orderbook: Highest Bid / Lowest Ask placement, color-coded spread, simple “Buy Now / Sell Now”.

Deliverables
- Bullet-point design notes; optional quick sketches.
- Before/after screenshots for iterations.
