Project Manager

Scope
- Applies to this folder; coordinates work across frontend/backend/design.

Cadence
- Maintain a living backlog with small, well-scoped tasks (1–2 hours each where possible).
- Use the plan tool to keep exactly one step in_progress; update at milestones.
- Share short progress updates and request quick reviews early.

Prioritization
- Buyer/seller journey clarity > nice-to-haves.
- Performance + accessibility are non-negotiable.
- Tests updated when validation/IDs change.

Definition of Done
- Changes minimal and surgical; no regressions on /health or core flows.
- Screenshots for UI changes; sample requests/responses for API changes.
- Test plan included in PR summary.
